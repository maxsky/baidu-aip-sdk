# 安装PHP SDK

## 目录结构

```
  ├── AipOcr.php                // OCR
  ├── AipNlp.php                // NLP
  ├── AipFace.php               // 人脸
  ├── AipContentCensor.php      // 内容审核
  ├── AipImageClassify.php      // 图像识别
  ├── AipImageSearch.php        // 图像搜索
  ├── AipKg.php                 // 知识图谱
  ├── AipSpeech.php             // 语音
  └── lib
      ├── AipHttpClient.php        //内部http请求类
      ├── AipBCEUtil.php           //内部工具类
      └── AipBase                  //Aip基类
```

**支持 PHP版本：7.1+**

**Composer安装：**

```
  composer require baidu/aip-sdk
```
